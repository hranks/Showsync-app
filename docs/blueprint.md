# **App Name**: DJ Ledger

## Core Features:

- Upcoming Gig Alert: Display a 'Next Event' bubble that shows the closest upcoming gig.
- Performance Dashboard: Dashboard providing monthly income summaries, hours worked, and venue/event type breakdown.
- Manual Event Logging: Enable manual entry of events including date, time, hours worked, total earnings, event type (Club/Corporate), venue, and optional notes.
- Recurring Venue Profiles: Provide pre-defined frequent venues with fixed hourly rates.
- Automated Wage Calculation: Management fixed hourly rates depending on the venue ($15, $20, $25), plus overtime pay calculation at $10/hour.
- Event Segregation: Separate tabs or filters for past events, upcoming events, and earnings summaries.

## Style Guidelines:

- Primary color: A vibrant purple (#9D4EDD), reminiscent of club lighting, provides a modern and energetic feel.
- Background color: A dark gray (#212529), offering a sleek, modern aesthetic conducive to both light and dark modes and provides a high contrast base for content.
- Accent color: An electric green (#52B788) to highlight key actions and information, drawing the user's attention to critical data points.
- Body and headline font: 'Inter' (sans-serif) provides a clean, modern, and easily readable typeface suitable for both body text and headings, ensuring clarity across the interface.
- Use category colors or icons to visually differentiate between Clubs and Corporate events at a glance.
- Maintain a clean and minimal layout optimized for fast input and easy viewing, ensuring clarity and ease of navigation.
- Employ subtle animations when transitioning between views to maintain user engagement without causing distraction.
